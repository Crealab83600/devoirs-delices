Projet personnel – Devoirs & Délices

À 16 ans, j’ai créé un site web complet de soutien scolaire bénévole pour les élèves de collège et de lycée à Fréjus.

Technologies utilisées :

  • HTML (plus de 2000 lignes)
  • CSS (design responsive type plateforme professionnelle)
  • JavaScript (calendrier interactif, statistiques, gestion des réservations)
  • GitHub Pages pour l’hébergement

Fonctionnalités :

  • Calendrier mensuel interactif
  • Système de réservation
  • Page d’administration
  • Statistiques en temps réel
  • Interface utilisateur professionnelle

Ce projet m’a permis de développer mon autonomie, mes compétences techniques en développement web et mon engagement bénévole.
