Bienvenue sur notre site pédagogique

Nous proposons des cours, exercices et ressources pour les élèves de collège et lycée. Vous pouvez consulter les matières, poser vos questions et prendre rendez-vous pour des aides personnalisées.

Mathématiques

Algèbre : Résolution d'équations

Pour résoudre une équation : isoler l’inconnue et vérifier la solution.

  • Exemple : 3x + 5 = 17 → x = 4
  • Exercice : 2x - 7 = 13 → x = ?
  • Exercice : 5y + 12 = 32 → y = ?

Géométrie : Triangles et angles

La somme des angles d’un triangle = 180°.

  • Exercice : Deux angles = 50° et 60°, trouver le troisième → 70°
  • Exercice : Triangle équilatéral, chaque angle ? → 60°
  • Exercice : Triangle isocèle, angles égaux = 45°, angle restant ? → 90°

Probabilités et statistiques

  • Exercice : Tirer un as dans un jeu de 52 cartes → 1/13
  • Exercice : Probabilité d’un nombre pair sur un dé → 3/6 = 1/2
  • Exercice : Lancer deux dés, probabilité d’avoir une somme = 7 → 6/36 = 1/6

Exercices supplémentaires

  • Factoriser x² + 5x + 6 → (x+2)(x+3)
  • Résoudre 4x - 9 = 7 → x = 4
  • Somme des angles quadrilatère → 360°

Français

Grammaire : Les accords

  • Les verbes s’accordent avec le sujet
  • Les adjectifs s’accordent avec le nom
  • Exercice : Les filles (être) contentes → Les filles sont contentes
  • Exercice : Le garçon (avoir) fini → Le garçon a fini

Orthographe : Les homophones

  • et / est, a / à, son / sont
  • Exercice : Il (a/à) une pomme → Il a une pomme
  • Exercice : Les chats (son/sont) mignons → Les chats sont mignons

Rédaction : Construire un paragraphe

  • Phrase d’introduction
  • Développement avec exemples
  • Conclusion ou phrase finale
  • Exercice : Rédigez un paragraphe sur votre loisir préféré

Lecture et analyse de texte

  • Identifier thème, personnages, actions principales
  • Analyser un extrait de "Le Petit Prince"
  • Exercice : Résumez chaque chapitre en 3 phrases
