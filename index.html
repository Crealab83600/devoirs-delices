Articles Pédagogiques (1-10)

Article 1 : Résoudre des équations du premier degré

Méthode : Isoler l’inconnue en utilisant les opérations inverses.

Exemple : 2x + 3 = 11 → 2x = 8 → x = 4.

Exercice : 3x - 5 = 10. Solution : x = 5.

Article 2 : Calculer la moyenne et l’écart-type

Méthode : Moyenne = somme / nombre de valeurs, écart-type pour dispersion.

Exemple : Notes : 12, 15, 18 → moyenne = 15.

Exercice : Notes : 10, 14, 16. Solution : Moyenne = 13.33.

Article 3 : Identifier les thèmes principaux d’un texte

Méthode : Lire attentivement, souligner idées répétées, résumer.

Exemple : Texte sur le climat → thème : réchauffement climatique.

Exercice : Résume un texte sur la biodiversité. Solution : disparition d’espèces, importance des écosystèmes.

Article 4 : Factoriser une expression simple

Méthode : Identifier un facteur commun et le mettre en évidence.

Exemple : 6x + 9 = 3(2x + 3).

Exercice : 8a + 12. Solution : 4(2a + 3).

Article 5 : Développer une expression

Méthode : Appliquer la distributivité.

Exemple : 3(x + 4) = 3x + 12.

Exercice : 5(y + 2). Solution : 5y + 10.

Article 6 : Travailler sur les proportions

Méthode : Établir un rapport, multiplier/diviser par le même facteur.

Exemple : 4/5 = x/20 → x = 16.

Exercice : 3/7 = x/14. Solution : x = 6.

Article 7 : Lire un diagramme en bâtons

Méthode : Identifier l’axe des valeurs, lire les hauteurs des barres.

Exemple : Graphique : 3 élèves aiment maths, 5 aiment français.

Exercice : Interpréter un graphique sur les sports préférés. Solution : Football = 40%, Tennis = 30%, Natation = 30%.

Article 8 : Identifier le COD et COI

Méthode : Le COD répond à “qui/quoi ?”, le COI à “à qui/à quoi ?”.

Exemple : Elle écrit une lettre → COD = lettre.

Exercice : Pierre parle à son ami. Solution : COI = son ami.

Article 9 : Résoudre une équation du second degré simple

Méthode : Utiliser Δ et factoriser si possible.

Exemple : x² - 5x + 6 = 0 → (x-2)(x-3)=0 → x=2 ou x=3.

Exercice : x² - 7x + 12 = 0. Solution : (x-3)(x-4)=0 → x=3 ou x=4.

Article 10 : Calculer un pourcentage

Méthode : Pourcentage = (partie / total) × 100.

Exemple : 15 sur 60 → 25%.

Exercice : 12 sur 50. Solution : 24%.

Articles Pédagogiques (11-30)

Article 11 : Factoriser une expression simple

Méthode : Identifier un facteur commun et le mettre en évidence.

Exemple : 6x + 9 = 3(2x + 3)

Exercice : 8a + 12. Solution : 4(2a + 3)

Article 12 : Développer une expression

Méthode : Appliquer la distributivité.

Exemple : 3(x + 4) = 3x + 12

Exercice : 5(y + 2). Solution : 5y + 10

Article 13 : Calculer une moyenne pondérée

Méthode : Multiplier chaque note par son coefficient, additionner, puis diviser par la somme des coefficients.

Exemple : Notes : 12 (coef 2), 14 (coef 3) → (12*2 + 14*3)/5 = 13.6

Exercice : Notes : 10 (coef 3), 15 (coef 2). Solution : 12

Article 14 : Résoudre une équation avec parenthèses

Méthode : Développer puis isoler l’inconnue.

Exemple : 2(x + 3) = 10 → 2x + 6 = 10 → 2x = 4 → x = 2

Exercice : 3(y - 2) = 9. Solution : y = 5

Article 15 : Identifier le sujet et le verbe

Méthode : Le sujet fait l’action, le verbe décrit l’action.

Exemple : Le chat dort → sujet = Le chat, verbe = dort

Exercice : Marie mange une pomme. Solution : Sujet = Marie, verbe = mange

Article 16 : Utiliser les angles dans un triangle

Méthode : La somme des angles = 180°, calculer l’inconnu.

Exemple : A=50°, B=60° → C = 70°

Exercice : A=40°, B=65°. Solution : C = 75°

Article 17 : Calculer un pourcentage

Méthode : Pourcentage = (partie / total) × 100

Exemple : 15 sur 60 → 15/60*100 = 25%

Exercice : 12 sur 50. Solution : 24%

Article 18 : Résoudre un système de 2 équations simples

Méthode : Méthode de substitution ou addition.

Exemple : x + y = 10, x - y = 4 → x = 7, y = 3

Exercice : 2a + b = 9, a - b = 1. Solution : a=2.5, b=4

Article 19 : Identifier le COD et COI

Méthode : COD = qui/quoi, COI = à qui/à quoi

Exemple : Elle écrit une lettre → COD = lettre

Exercice : Paul parle à sa sœur. Solution : COI = sa sœur

Article 20 : Analyser un graphique linéaire

Méthode : Lire les axes, interpréter les tendances.

Exemple : Croissance ventes janvier → février = +20%

Exercice : Courbe température sur 5 jours. Solution : Augmentation de 10°C à 20°C

Article 21 : Lire un tableau de données

Méthode : Identifier lignes, colonnes et valeurs clés.

Exemple : 5 élèves ont 12, 3 ont 15 → calculer moyenne

Exercice : Notes : 10, 12, 15, 18. Solution : Moyenne = 13.75

Article 22 : Identifier le temps des verbes

Méthode : Observer terminaison et auxiliaire.

Exemple : J’ai mangé → passé composé

Exercice : Ils partiront. Solution : Futur simple

Article 23 : Résoudre un problème de proportion

Méthode : Établir rapport, multiplier/diviser par facteur.

Exemple : 4/5 = x/20 → x = 16

Exercice : 3/7 = x/14. Solution : x=6

Article 24 : Identifier le type de phrase

Méthode : Interrogative, déclarative, impérative, exclamative.

Exemple : “Va à l’école !” → impérative

Exercice : “Quelle heure est-il ?” Solution : Interrogative

Article 25 : Utiliser la distributivité double

Méthode : (a+b)(c+d) = a*c + a*d + b*c + b*d

Exemple : (x+2)(x+3) = x² +5x +6

Exercice : (a+1)(a+4). Solution : a²+5a+4

Article 26 : Identifier le sujet inversé

Méthode : Sujet après le verbe → phrase interrogative ou inversion.

Exemple : “A-t-il fini ?” → sujet = il

Exercice : “Sont-ils partis ?” Solution : Sujet = ils

Article 27 : Utiliser la règle de 3

Méthode : a/b = x/c → x = b*c / a

Exemple : 4/8 = x/32 → x = 16

Exercice : 5/10 = x/50. Solution : x=25

Article 28 : Résoudre un problème de vitesse

Méthode : v = d/t, d = v*t, t = d/v

Exemple : Voiture 60 km/h, distance 120 km → t=2h

Exercice : V=50 km/h, d=150 km. Solution : t=3h

Article 29 : Identifier le complément circonstanciel

Méthode : Répond à où, quand, comment, pourquoi.

Exemple : “Il part demain.” → CC = demain

Exercice : “Elle court dans le parc.” Solution : CC = dans le parc

Article 30 : Résoudre un problème de périmètre et aire

Méthode : Formules périmètre = somme côtés, aire selon forme

Exemple : Rectangle 5x3 → périmètre 16, aire 15

Exercice : Carré côté 4. Solution : P=16, A=16

Articles Pédagogiques (31-50)

Article 31 : Résoudre un système par substitution

Méthode : Isoler une variable dans une équation puis remplacer dans l’autre.

Exemple : x + y = 7, y = 3 → x = 4

Exercice : 2a + b = 10, b = 4. Solution : a = 3

Article 32 : Calculer le volume d’un cube et d’un pavé

Méthode : Cube : côté³, pavé : L × l × h

Exemple : Cube côté 3 → 3³ = 27

Exercice : Pavé L=4,l=3,h=2. Solution : 24

Article 33 : Identifier les temps composés

Méthode : Verbe auxiliaire + participe passé

Exemple : J’ai fini → passé composé

Exercice : Nous avons joué. Solution : Passé composé

Article 34 : Travailler sur les fractions

Méthode : Addition : même dénominateur, sinon trouver PPCM

Exemple : 1/4 + 2/4 = 3/4

Exercice : 2/3 + 1/6. Solution : 5/6

Article 35 : Calculer une vitesse moyenne

Méthode : v = distance / temps

Exemple : 120 km en 2h → 60 km/h

Exercice : 90 km en 1,5 h. Solution : 60 km/h

Article 36 : Identifier le complément d’objet indirect

Méthode : Répond à “à qui / à quoi”

Exemple : Il parle à son ami → COI = son ami

Exercice : Elle téléphone à sa mère. Solution : COI = sa mère

Article 37 : Résoudre une inéquation simple

Méthode : Isoler l’inconnue, changer le sens si on multiplie/divise par un négatif

Exemple : 2x + 3 > 7 → 2x > 4 → x > 2

Exercice : 3y - 5 ≤ 7. Solution : y ≤ 4

Article 38 : Lire un graphique circulaire

Méthode : Identifier les parts et pourcentages

Exemple : 25% de 40 élèves aiment maths → 10 élèves

Exercice : 20% de 50 élèves aiment sport. Solution : 10 élèves

Article 39 : Identifier les adjectifs qualificatifs

Méthode : Modifient un nom ou un pronom

Exemple : Une maison grande → grande = adjectif

Exercice : Un film intéressant. Solution : intéressant = adjectif

Article 40 : Calculer l’aire d’un cercle

Méthode : Aire = π × r²

Exemple : Rayon = 3 → 9π

Exercice : Rayon = 5. Solution : 25π

Article 41 : Résoudre un problème de proportionnalité

Méthode : a/b = x/c → x = b*c / a

Exemple : 3/5 = x/20 → x = 12

Exercice : 4/7 = x/21. Solution : x = 12

Article 42 : Identifier le complément circonstanciel de temps

Méthode : Répond à “quand ?”

Exemple : “Il part demain” → CC temps = demain

Exercice : “Elle reviendra lundi.” Solution : lundi

Article 43 : Décomposer un nombre en facteurs

Méthode : Trouver les nombres premiers

Exemple : 12 = 2 × 2 × 3

Exercice : 18. Solution : 2 × 3 × 3

Article 44 : Identifier la fonction grammaticale

Méthode : Sujet, COD, COI, CC selon rôle dans la phrase

Exemple : Il lit un livre → sujet = Il, COD = livre

Exercice : Marie donne un cadeau à Paul. Solution : sujet=Marie, COD=cadeau, COI=Paul

Article 45 : Résoudre un problème d’échelle

Méthode : distance réelle = distance sur plan × échelle

Exemple : Échelle 1:100 → 5 cm → 5 m

Exercice : 8 cm sur plan, échelle 1:200. Solution : 16 m

Article 46 : Utiliser la formule de l’aire d’un triangle

Méthode : Aire = (base × hauteur)/2

Exemple : Base 6, hauteur 4 → 12

Exercice : Base 10, hauteur 5. Solution : 25

Article 47 : Résoudre un problème de moyenne

Méthode : Somme des valeurs / nombre de valeurs

Exemple : 12, 14, 16 → 14

Exercice : 10, 15, 20. Solution : 15

Article 48 : Identifier les adverbes

Méthode : Modifient le verbe, adjectif ou autre adverbe

Exemple : Il court vite → vite = adverbe

Exercice : Elle parle doucement. Solution : doucement = adverbe

Article 49 : Calculer le périmètre d’un cercle

Méthode : P = 2 × π × r

Exemple : r=3 → P=6π

Exercice : r=5. Solution : P=10π

Article 50 : Identifier les mots invariables

Méthode : Ne changent pas selon genre, nombre ou personne

Exemple : hier, très, mais

Exercice : Trouve les invariables : “Il court vite mais heureusement.” Solution : vite, mais, heureusement
